# Dataset Mapping & Query

This section covers how to map datasets to different dimension conventions and
how to query dsgrid data.

**Mapping** transforms data from one dimension definition to another (e.g., counties
to states, or interpolating between years).

**Querying** extracts, filters, and aggregates data from registered datasets or
projects.
