# Projects

This section is for **project coordinators** who create and manage dsgrid projects.

A dsgrid project is a collection of datasets describing energy demand for a specific
region and timeframe. Project coordinators:

- Create and configure projects
- Define base and supplemental dimensions
- Coordinate with dataset contributors
- Run queries and create derived datasets
- Publish data for users
