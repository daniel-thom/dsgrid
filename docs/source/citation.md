# Citation and Attribution

This page provides guidance on citing dsgrid and attributing data.

## Citing dsgrid

If you use dsgrid software or data in your work, please cite it appropriately.

### Software Citation

For citing the dsgrid software:

> dsgrid: Demand-Side Grid Toolkit. National Renewable Energy Laboratory.
> https://github.com/dsgrid/dsgrid

### Data Citation

When citing specific datasets, include:

- Dataset name and version
- Original data source/creator
- Access date
- DOI (if available)

Example citation for TEMPO data:

```text
TEMPO County-level EV Charging Profiles. National Renewable Energy Laboratory.
Available at: s3://nrel-pds-dsgrid/tempo/
```

## Data Attribution

dsgrid datasets come from various sources. When using published data, ensure you:

1. Check the dataset's origin information in its configuration
2. Attribute the original data creators
3. Note any usage restrictions or licenses

### Original Data Sources

Common data sources in dsgrid projects include:

- **ResStock**: NREL residential building stock model
- **ComStock**: NREL commercial building stock model
- **TEMPO**: NREL Transportation Energy & Mobility Pathway Options
- **AEO**: EIA Annual Energy Outlook

Consult each dataset's documentation for specific attribution requirements.
